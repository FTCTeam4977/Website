LiveFeed

LiveFeed was created to make it easy to display "live" updates in a feed. The "click me" button demonstrates key features of LiveFeed. By clicking it, you cycle through a switch that calls the "update" function with different arguments. Follow these steps to begin using live feed.

  1. In your body tag, add the "onload" event. As a string, give it the "start" function. "Start" can take up to four arguments: location, height, width, and scroll, but scroll is not required. Ex "<body onload="start('cat', 400, 200);">".
  2. Create a blank Div that doesn't contain anything with the id of the location you specified in the start function. Ex "<div id="cat"></div>"
  3. To add a Feed, you need to call the "update" function. "Update" takes three arguments:location, height, and content. Ex "update("cat",40,"This is the first post");".

Things to note

The feed needs a minimum amount of space to work so height can't be set to below 30 and width can't be set to below 400.
To customize the theme of LiveFeed, edit the variables in the LiveFeed.js file in the customizable section.
